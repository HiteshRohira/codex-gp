# Icons

Place SVG icons for the GPUI desktop shell in this directory.

Needed icons:

- `new-chat.svg` - create a new thread.
- `folder-plus.svg` - add or choose a project folder.
- `folder.svg` - project row.
- `message-square.svg` - thread row.
- `search.svg` - sidebar search field.
- `paperclip.svg` - attach context from the composer.
- `send.svg` - send the composer message.
- `stop-circle.svg` - interrupt an active turn.
- `terminal.svg` - command/tool output rows.
- `file-diff.svg` - file change/diff rows.
- `brain.svg` - reasoning/thinking rows.
- `check-circle.svg` - completed state.
- `alert-circle.svg` - approval or warning state.
